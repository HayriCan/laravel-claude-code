# Laravel Brainstorming

Start an interactive design session for a new Laravel feature.

## Usage

```
/brainstorm
/brainstorm user authentication with social login
/brainstorm blog post management with categories and tags
/brainstorm payment integration with Stripe
```

## Workflow

1. **Understand** - Clarify requirements through questions
2. **Explore** - Propose 2-3 architecture approaches
3. **Design** - Present design in digestible sections
4. **Validate** - Get approval for each section
5. **Document** - Save to `docs/plans/YYYY-MM-DD-<feature>-design.md`

Invoke the laravel-brainstorming skill and follow it exactly as presented.

**Feature description:** $ARGUMENTS
